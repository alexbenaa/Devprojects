Tags HTML


Tags de Estrutura

!DOCTYPE html -> Indica para o navegador que a página HTML iniciou.
html -> Inicia a estrutura o html
head -> Cabeçalho da página, coisas que não estão no corpo
body -> Corpo da estrutura / site
meta charset="utf-8"> -> Sistema ASCII de escrita, definida como utf-8

Tags de titulos/subtitulos

h1 -> Titulo principal
h2 -> Subtitulos
h3 -> Menor importância e por ai vai..... até h6
p -> Tag responsável por criar um paragráfo.
b -> Texto em Negrito. "b"de Bold
strong -> Também deixa em negrio porém, informa aos sistemas de busca, que este texto é mais importante
i -> Texto em Itálico
em -> Texto em itálico porém com um realse nos localizadores como o
mark -> Texto com macações padrões do HTML
del -> Risca o texto
ins -> Texto sublinhado como se fosse inserido
sub -> Texto subscrip
sup -> Texto Superscrip
img src="" alt="" -> Comando utilizado para adicionar Imagens.
Os comandos width=largura, heigth=altura da imagem
br -> Tag utilizada para quebra de linha, pode ser utilizado várias vezes.

Listas em HTML


"ul" -> Tag utilizada para criar uma lista, unordenerlist.
"ol" -> Tag utilizada para criar uma lista ordenada
"li" -> Tag para adicionar um item a lista

  • Teste do item 1
  • Teste do item 4
Exemplo de lista ordenada

  3. Teste do item 1
  4. Teste do item 4
